# 비교 연산자 & 판단 명령어 if / else
```python
  a = 3

if a == 2:
    print("A")

if a == 3:
    print("B")

if a == 4:
    print("C")
else:
    print("D")
```
## 비교 연산자
   - == : 양쪽이 같다. (같으면 True, 다르면 False)
   - != : 양쪽이 다르다. (다르면 True, 같으면 False)

## 판단 명령어 if / else
    - if 비교할 문장:
         True일 때 실행할 문장
      else
         False일 때 실행할 문장
