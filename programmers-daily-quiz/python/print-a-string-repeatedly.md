### 알게된 것
① map( )
  - map(함수, 반복 가능한 객체)
  - list나 tuple같은 data의 모든 요소에 특정 함수를 한번에 적용

② strip( )
  - 문자의 양 끝에서 공백이나 특정 문자를 제거.
  - rstrip( ), lstrip( )으로 쓰면 한쪽만 지우는 것도 가능.

③ split( )
  - 문자열을 일정한 규칙으로 쪼개어 list 형태로 반환
  - data를 parsing하거나 쉼표, 띄어쓰기 등으로 구분된 값을 추출할 때 매우 유용함

④ lan( )
  - ( )안의 data가 몇개의 요소로 이루어져 있는지 개수를 카운터해 숫자로 알려줌.
  - 문자열, list 등 다 사용가능
