## 알게된 것

① 대소문자 변환 주요 메서드
- .upper( ) : 모든 문자를 대문자로 변환
- .lower( ) : 모든 문자를 소문자로 변환
- .swapcase( ) : 대문자 -> 소문자, 소문자 -> 대문자 반전
- .capitalize( ) : 첫 글자만 대문자로, 나머지는 소문자로 변환
- .title( ) : 각 단어의 첫 글자를 대문자로 변환

② 뒤에 ( )가 붙는 이유
- ( )는 이 기능이 단순히 값을 가져오는 것이 아니라 지금 실행하라고 명령을 내리는 호출 기호이기 때문
- 비유 : 메서드 = 리모컨 버튼, ( ) = 버튼을 누르는 행위

③ 문자열 메서드 중 ( ) 안에 추가 정보를 넣어줘야 하는 것들
- replace(찾을 문자, 바꿀 문자) : 어떤 문자열을 다른 문자로 바꿀 때
- count(찾을 문자) : 특정 문자가 몇 번 나오는지 알려줌
- split(기준 문자) : 문자를 쪼개서 알려

④ ( )안에 " "를 쓰는 경우와 그렇지 않은 경우
- 쓰는 경우 : 문자(텍스트)를 직접 쓸 때
- 쓰지 않는 경우 : 숫자를 넣을 때, 변수를 넣을 때, 다른 함수나 식을 넣을 때
