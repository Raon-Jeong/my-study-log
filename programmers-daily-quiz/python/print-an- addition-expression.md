# 덧셈식 출력하기
## 문제
```
Q. 두 정수 a, b가 주어질 때 다음과 같은 형태의 계산식을 출력하는 코드를 작성해 보세요.
   a + b = c

※ 제한 사항 ※
  - 1 <= a, b <= 100

※ 입출력 예 ※
   입력 #1
   4 5
   출력 #1
   4 + 5 = 9
```
## 준비된 코드
```python
a,b = map(int, input().strip().split(' '))
print(a + b)
```
## 나의 풀이
```python
a, b = map(int, input( ).strip( ).split(' '))
if 1 <= a <= 100 and 1 <= b <= 100 :
  c = a + b
print(f"{a} + {b} = {c}")
```
## 다른 사람 풀이
```python
a, b = map(int, input( ).strip( ).split(' '))
if 1 <= a <= 100 and 1 <= b <= 100 :
print(f"{a} + {b} = {a + b}")

# 첫번째 줄에서 a, b가 숫자로 되었기 때문에 별도로 C를 만들 필요없이 그냥 바로 a + b를 해도 됨
```
## 배운 것
- input( ).strip( ).split(' ')

  : 사용자로부터 한 줄의 문자열을 입력받아, 공백을 기준으로 쪼개서 리스트를 만드는 코드

- map(적용할 함수, 반복 가능한 데이터)

  : 어떤 리스트나 묶음 안의 모든 항목에 특정 함수를 한번에 똑같이 적용해주는 유용한 함수.
