# 확률과 통계
## 기초 통계
### 기초 통계
 - 대량의 데이터를 숫자로 정리 및 분석하는 방법
 - 데이터가 많아질수록 대푯값을 찾는 것이 중요
 - 기초통계의 대푯값
   ① 평균값(mean) : 모든 자료의 합 / 자료의 개수
     - 이상치의 영향을 많이 받음
   ② 중앙값(median)
     - 자료를 크기순으로 나열했을 때 가운데 값
     - 이상치의 영향을 거의 받지 않음
   ③ 최빈값(mode)
     - 가장 자주 나타나는 값
     - 범주형 자료에서도 사용 가능
 - 기초통계의 산포도 : 대푯값이 전체 자료를 얼마나 잘 대표하는지 판단할 때 사용
   ① 사분위수(IQR:Interquartile Range)
     - 자료 분포를 4등분(Q1:25%, Q2:50%, Q3:75%)해서 중간군 퍼짐 보기
     - Q3 - Q1 => 중간 50% 자료가 얼마나 퍼져 있는지 보여주는 범위
   ② 분산
     - 평균으로부터의 편차를 '제곱'해서 평균낸 값
     - 퍼짐의 크기를 계산한 값
     - 평균에서 얼마나 멀리 떨어져 있는가를 제곱한 평균값
     - 값이 클수록 자료가 넓게 퍼짐
     - 값이 실제보다 과도하게 커질 수 있음.
   ② 표준편차
     - 분산의 제곱근에 루트를 씌워 => 원래 데이터로 단위를 바꿔서 분포의 퍼진 정도를 직관적으로 보여줌
     - 평균으로부터 얼마나 떨어져 있는지 나타내는 거리
     - 퍼짐을 거리로 이해할 수 있게 만든 값
### 확률 분포
 - 확률의 개념
   - 각 값이 나타날 가능성을 수치화
 - 확률 기초
   - 기초 통계 : 이미 발생한 데이터를 정리 및 분석하는 것
   - 확률 : 이미 발생한 데이터를 바탕으로 새로운 데이터를 예측
 - AI와 확률
   - AI는 Softmax라는 방식을 사용하여 확률을 예측 => 정답을 고르는게 아닌 가능성을 확률로 나눠서 제시하며 예측
   - ex) 확률 기반 알고리즘
