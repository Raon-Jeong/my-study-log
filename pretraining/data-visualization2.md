# 시각화 라이브러리
## Matplotlib
 - 파이썬에서 가장 기본이 되는 시각화 도구
 - 거의 모든 데이터 분석의 출발점
 - 단순하지만 원리 이해에 강점
## Seaborn
 - Matplotlib을 기반으로 만들어진 고급 시각화 도구
 - 같은 데이터라도 더 예쁘고, 더 직관적으로 표현
 - 통계적 시각화에 강점

# 탐색적 데이터 분석(EDA)
 - Exploratory Data Analysis
 - 데이터에 숨겨진 패턴, 이상치, 변수 간 관계 등을 발견하기 위해
   요약 통계와 시각화 도구를 사용하는 데이터 조사 과정
 - 탐색적 데이터 분석을 통해 데이터셋을 다각도로 관찰하여 새로운 정보나 잠재적 문제를 발견하고
   향후 분석 방향을 설정하는 데 도움
## 상관관계
 - 두 변수가 함께 변하는 통계적 연관성
 - 방향에 따라 양의 상관관계(+), 음의 상관관계(-)를 가짐
 - 상관계수 값의 절댓값이 1에 가까울수록 강하고, 0에 가까울수록 약함(0은 관계 없음)
