## 선생님 풀이
```python
while True :
    n1 = int(input("첫 번째 정수 입력 >>"))
    n2 = int(input("두 번째 정수 입력 >>"))
    if n1 == 0 and n2 == 0 :
        print("프로그램이 종료되었습니다.")
        break
    print("두 정수의 합 : ", n1 + n2)
```

## 배운것
  ### - while
   - 조건식이 True일 때 실행문장 반복
   - 반복 횟수가 명확하지 않을 때 사용
   - while True => 무한 반복
   - break : 반복문을 나가는 명령어
  ### - for
   - 문자열이나 list, tuple이 들어갔을 때 안에 있는 요소를 하나식 반복
   - 반복 횟수가 명확할 때 사용
  ### - len( )
   - 데이터의 개수나 길이를 세어주는 함수
   - ( ) 안에 리스트, 문자열, 튜플 등을 넣으면 그 안에 요소가 총 몇 개 들어있는지 숫자로 알려줌.
  ### - range( )
   - range(시작할 숫자, 종료할 숫자+1, 증가량)
   - 필요한 만큼의 숫자를 만들어내는 기능
  ### - 반복해서 나와야 하는건 반복문 안에 넣어야 함. => 그래야 코드가 깔끔
