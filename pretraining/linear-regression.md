# 선형회귀
## 선형회귀
 - 원인 변수와 결과 변수 사이의 관계를 하나의 직선으로 표현하여 데이터의 전반적인 패턴을 이해하려는 통계 기법
## 선형회귀 식
 - 원인(문제) 데이터 : 독립변수, x값
 - 결과(답) 데이터 : 종속변수, y값
 - y = wx + b
## 선형회귀의 목표
 - 모든 선을 대표하는 직선 찾기
## 잔차
 - 실제 값과 선이 예측한 값의 차이
 - 기계의 성능이 좋은지 안 좋은지를 판단하는 객관적인 기준
 - 잔차 = 실제값 - 예측
 - 잔차 = H(x) - y    =>   H(x) : 예측 값, y : 원 데이터
```python
# 몸무게
w = [50, 55, 60, 65, 70]

# 실제 키
h = [160, 165, 170, 172, 175]

# 예측된 키를 저장할 예측 값을 담는 리스트
pred = []

for i in w :    # 몸무게
    p = i + 115     # y = wx + b
    pred.append(p)

print(pred)
-------------------------------------------------------------------------------------
[165, 170, 175, 180, 185]
```
```python
# 몸무게
w = [50, 55, 60, 65, 70]

# 실제 키
h = [160, 165, 170, 172, 175]

pred = []

    p = i + 115     # y = wx + b
    pred.append(p)

# 잔차를 구해서 저장할 리스트
res = [ ]

for i in range(5) :
    r = h[i] - pred[i]   # 실제 키 - 기계가 예측한 키 => 잔차
    res.append(r)

for i  in range(5) :
    print("몸무게 :", w [i])
    print("실제 키 :", h [i])
    print("예측 키 :", pred [i])
    print("잔차 :", res [i])
    print( )
```
