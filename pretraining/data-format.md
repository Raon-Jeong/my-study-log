# 데이터 형식
 - 데이터를 저장하고 처리할 때 그 크기와 표현 방식을 정의하는 것
## 정형 vs 비정형
### 정형 데이터
 - 행과 열이 정해진 표 형태 데이터
 - 구조가 명확함
 - 바로 계산 및 분석 가능
 - Excel, CSV, SQL DB
### 비정형 데이터
 - 형식이 정해져 있지 않은 데이터
 - 사진, 영상, 음성 등 형식이 없음
 - 정형 데이터로 전처리 필요
 - NLP, CV, 딥러닝 모델
## 시계열 데이터
 - 시간의 흐름에 따라 순서대로 기록된 데이터
 - "언제 발생했는가"가 중요한 데이터
 - 순서가 바뀌면 의미가 완전히 바뀌는 데이터
 - 시간 간격이 존재하는 데이터 (매일, 매월, 매시간 등)
 - 시간에 따른 변화 추이를 관찰하고 예측할 때 사용
## 미디어 데이터
 - 이미지, 영상, 음성과 같이 사람이 보고, 듣는 형태의 데이터
 - 글자나 숫자가 아닌 감각 정보를 담은 데이터
 - 이미지 데이터 : 픽셀이라는 숫자값 들의 집합
 - 영상 데이터 : 여러 장의 이미지가 시간 순서대로 연결된 데이터
 - 음성 데이터 : 소리의 크기와 파형을 숫자로 기록한 데이터

# 표 데이터 접근
## DataFrame 내부 데이터 접근
### DataFrame은 순서가 있는 데이터
 - 인덱스가 있음 -> 인덱싱 or 슬라이싱이 가능함 -> 표에 갇혀진 데이터인 DataFrame에서는 도구가 필요 -> loc, iloc 도구를 사용
