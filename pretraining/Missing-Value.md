# 결측치
 - 비어있는 값
 - 프로그래밍에서 비어있는 값은 Null이라고 표현함
 - 보통 주변 데이터를 보고 비어있는 값을 유추함
 - 대표값으로 결측치를 대체하거나, 결측치가 포함된 인스턴스 삭제
 - 각종 연산 불가능
 - 계산하는 데 엄청난 방해 요소

# 이상치
 - 다른 값들과 너무 동떨어진 값
 - 영어로 outlier(문외한)이라고 부름
 - 보통 주변 데이터에 비해 눈에 띄게 특이한 값을 가지고 있음
 - 표준 편차나 그래프 시각화를 이용해 탐지
 - 각종 통계 연산 및 분석 결과 왜곡
 - 결측치나 이상치를 가지고 계산하는 것은 위험함

# 데이터 전처리
 - 데이터를 분석 및 학습에 적합한 상태로 변환하는 과정
 - 데이터 정제, 통합, 변환, 축소, 특성 공학 등을 행하는 것
 - 전체 데이터 품질 향상
 - 모델 성능 개선
 - 분석 효율성 증대

# 스케일링
 - 값 범위나 분포를 동일한 수준으로 조정하는 전처리 과정
## Standard Scaler
 - 변수의 평균, 분산을 이용해 정규분포 형태로 변환 (평균 0 , 분산 1)
 - 이상치에 민감하게 반응함
## MinMax Scaler
 - 데이터를 0~1사이 값으로 변환 (가장 작은 값 = 0, 가장 큰 값 = 1)
 - 이상치에 민감하게 반응함
