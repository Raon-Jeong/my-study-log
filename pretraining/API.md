# API(Application Programming Interface)
## 정의
 - 어플리케이션 간 상호작용을 하기 위한 인터페이스
 - 다른 프로그램과 대화할 수 있도록 돕는 통역사
 - 다양한 어플리케이션의 기능과 데이터를 간단한 인증을 통해 코드로 호출할 수 있도록 한 것
## 차이점
 - 파일 다운로드
   - 수동 다운로드
   - 최신 데이터 반영 어려움
 - API 사용
   - 코드로 요청
   - 항상 최신 데이터 다운로드 가능

# JSON / XMS / CSV
## JSON
 - 이름표(Key)가 붙은 데이터 묶음(dictionary와 유사함)
 - 구조 표현력이 탁월함
 - API 응담 최적화
 - CSV보다 용량이 큼
 - 초보자에겐 구조가 복잡함
## XML
 - 태그(</>)로 구성된 데이터
 - 구조, 의미가 명확
 - 복잡한 규칙 표현 가능
 - 길고 복잡함
 - 요즘 사용되지 않음
## CSV
 - 쉼표(,)로 구분된 데이터
 - 단순함
 - 저장 용량 작음
 - Pandas와 높은 호환
 - 데이터 설명 불가
## 공통점
 - 모두 다른 형식으로 구성되어 있지만, 최종목적지는 항상 "표 데이터"로 저장됨

# Pandas
## Pandas 라이브러리
- 파이썬에서 데이터 분석과 조작을 위해 개발된 대표적인 라이브러리
## DataFrame
- 행과 열을 가지는 표와 같은 형태 (2차원)
- 불러오는 법
```python
import pandas as pd
pd.DataFrame( )
```
