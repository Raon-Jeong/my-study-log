# numpy

## 배운 것
 ### - numpy
  - 대규모 다차원 배열을 효율적으로 다루는 과학 계산 라이브러리
 ### - numpy.ndarry 자료형
  - 동일한 자료형을 가지는 값들이 배열 형태로 존재 => 숫자는 숫자만, 글자는 글자만 써야 함
  - N차원 형태로 구성이 가능
  - 순서가 있는 집합
  - ndarray를 줄여서 array로 표현
```
# 1차원 numpy 자료형
  arr1 = na.array([1, 2, 3])
  arr1
  -----------------------------------------------------------------------------------------------------------
  array([1, 2, 3])

# 2차원 numpy 자료형
  arr2 = np.array([[1, 2, 3],
                   [4, 5, 6]])
  -----------------------------------------------------------------------------------------------------------
  array([[1, 2, 3],
         [4, 5, 6]])
```
 ### - 선형대수
   - 숫자 배열(벡터)을 이용해 계산하는 수학
   - AI는 글자, 이미지, 소리도 숫자로 바꿔서 처리
   - 이 숫자들을 계산하고 변형하는 언어가 선형대수
   - 선형대수에서는 행렬이라는 도구를 사용하여 연산
   - 이미지, 영상 인식 기술에 활용
```
< 벡터 >
 - 방향과 크기를 가지는 양
 - 하나의 대상을 여러 개의 숫자로 표현한 것 (숫자가 한 줄로 나열된 묶음)
 - 각 숫자는 대상의 특징(feature)
 - 벡터의 요소별 연산 : 벡터끼리의 크기가 같으면 같은 자리끼리 더하는게 가능함
 - 벡터의 차원 : [ ] 1차원, [[ ]] 2차원, [[[ ]]] 3차원
 - 벡터의 축(axis)
   ex) [[1, 2],
        [3, 4]]
       => 크기가 2(행) x 2(열)
< 행렬 >
 - 특정 벡터에 작용하는 힘(선형함수)
< 벡터의 연산>
 - 행렬과 벡터를 연산하기 위해서는 행렬과 벡터의 크기가 같아야 함
 - 벡터의 곱연산
   ex) 2 x [1,2,3] = [2,4,6]  => 스칼라 곱

        [[1, 2],  x  [[1, 2, 3], = [[1x1+2x4, 1x2+2x5, 1x3+2x6],
         [3, 4]]      [4, 5, 6]]    [3x1+4x4, 3x2+4x5, 3x3+4x6]]
           2x2            2x3               2x3
```
 ### - 이미지 데이터 처리
  - 세상의 많은 데이터는 이미지 or 영상으로 되어있음
  - 영상 데이터는 이미지 데이터에 '시간'이라는 차원이 더해져서 만들어짐
  - 우리가 카메라에서 사용하는 필터도 이미지 처리
  - 흑백 이미지 데이터 : 0 ~ 255 사이의 값으로 이루어진 거대 행렬
  - 컬러이미지 : RGB 채널(3개의 채널)이 겹쳐져서 만들어짐
  - RGB채널 : Red 행렬, Green 행렬, Blue 행
