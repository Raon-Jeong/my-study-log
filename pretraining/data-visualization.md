# 데이터 시각화
## 상관관계와 인과관계
### 상관관계
 - 두 데이터가 서로 관련되어 함께 변하는 현상
 - 상관관계가 있다고 해서 꼭 인과관계는 아닐 수 있음
### 인과관계
 - 어떤 원인이 생기면 필연적으로 특정 결과가 발생하는 것
### 데이터 편향(data bias)
 - 편향 : 어떤 생각이나 사물에 대해 한쪽으로 치우친 경향
 - AI 학습 데이터의 편향성의 영향으로 잘못된 패턴을 스스로 학습한 경우도 있음
## 데이터 시각화
### 산점도(Scatter plot)
 - 두 변수의 관계를 점의 모양과 방향으로 직관적으로 보여주는 그래프
 - 두 데이터 사이의 상관관계를 볼 때 사용
 - 관계가 있는지 없는지
 - 함께 늘거나 줄어드는 경향이 있는지
 - 데이터가 몰려있는지, 흩어져 있는지
```python
import matplotlib.pyplot as plt

# 예시 데이터
exercise = [0, 1, 2, 3, 4, 5, 6, 7]  # 주당 운동 시간
weight = [80, 78, 76, 74, 72, 71, 79, 69]  # 체중

# 산점도
plt.scatter(exercise.weight)

plt.show()
```
### 히스토그램(Histogram)
 - 하나의 변수 값이 구간별로 얼마나 많이 나타나는지를 막대로 보여주는 그래프
 - 데이터의 편향성을 확인할 때 사용
 - 값이 어느 구간에 많이 몰려있는지
 - 데이터가 넓게 퍼져 있는지, 좁게 모여 잇는지
 - 이상치의 존재여부
 - 분포가 한 쪽으로 치우쳐졌는지
```python
import matplotlib.pyplot as plt

# 예시 데이터
salary = [2200, 2400, 2600, 2800, 3000, 3200, 3400, 3600, 3800, 4500, 5000, 7000]   # 소수의 고액 연봉자

# 히스토그램
plt.hist(salary, bins = 6)

plt.show()
```
